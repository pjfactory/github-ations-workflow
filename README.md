## node github actions workflow 사용방법

### node-deploy

-   사용사는 yarn scripts:

    -   yarn config:download -s ${{ env.NODE_ENV }}
    -   yarn config:validate -s ${{ env.NODE_ENV }}
    -   yarn deploy:v1 -s ${{ env.NODE_ENV }}
